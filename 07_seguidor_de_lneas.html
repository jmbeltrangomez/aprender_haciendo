Robot siguelíneas

En esta práctica, se va a programar el robot para que se desplace por encima de una línea negra y la siga.

Se tiene que diseñar un circuito que disponga de una línea de, aproximadamente, 1,5 cm de ancho sobre un fondo blanco.

El algoritmo que se tendrá que realizar será el siguiente:

  1. Para siempre,
  2. Si se cumple la condición de que el sensor siguelíneas izquierdo y derecho detecta negro (es decir, un valor 0),
  3. los dos motores avanzan a una determinada velocidad,
  4. Si se cumple la condición de que el sensor siguelíneas izquierda detecta blanco (1) y el derecho negro (0),
  5. el motor izquierdo avanzará y el derecho se detendrá, provocando el giro del robot hacia la derecha, para de nuevo, detectar los dos sensores la línea negra,
  6. Si se cumple la condición de que el sensor siguelíneas izquierdo detecta negro (0) y el derecho blanco (1),
  7. el motor izquierdo se detendrá y el derecho avanzará, provocando el giro del motor hacia la izquierda, para de nuevo, detectar los dos sensores la línea negra y continuar avanzando.

Tatami

Este proyecto consiste en programar el robot para que no pueda abandonar un tatami, p.e. circular, delimitado por una línea negra. El robot al detectar la línea retrocederá y girará para buscar una nueva trayectoria que seguirá hasta que encuentre, de nuevo, una línea negra.

El algoritmo que se debe de realizar es el siguiente:

  1. Para siempre,
  2. Si el lector siguelíneas izquierdo y derecho detecta el color blanco (un valor 1),
  3. avanzará el robot,
  4. si no (y en este bloque se cumple alguna condición de que algún sensor siguelínea o los dos detectan negro (0)),
  5. el robot se parará,
  6. retrocederá un determinado tiempo, y
  7. girará hacia un lado,
  8. y de nuevo volverá a comenzar el proceso.

Contador de líneas

Otro proyecto que se puede plantear es el hacer que el robot maqueen cuente líneas.

El algoritmo que se debería de realizar es el siguiente:

  1. Definir una variable llamada "Contador"
  2. Al iniciar, establecer la variable "Contador" con un valor 0.
  3. Para siempre,
  4. Si la variable "Contador" tiene un valor igual a 2,
  5. los motores se detendrán,
  6. Si no,  se establecerá otro condicional
  7. Si algún siguelíneas (izquierda o derecha) detecta una valor negro,
  8. Cambiar la variable "Contador" por un 1 (es decir, sumarle al valor anterior un 1),
  9. Parar motores unos ms,
  10. Avanzar los motores unos pocos ms (para evitar que vuelva el contador a sumar 1 y así sucesivamente) para que salga de la línea,
  11. Si no detecta línea,
  12. Avanzar los motores hacia adelante.
